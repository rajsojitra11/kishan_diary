KISHAN DIARY - API REFERENCE (CURRENT)

Version: 2.0
Base URL: http://127.0.0.1:8000/api/v1
Auth: Bearer token for protected routes

============================================================
1) AUTH
============================================================

1. POST /auth/mobile-check
Body:
- mobile (required, 10 digits)

2. POST /auth/register
Body:
- name (required)
- email (nullable)
- mobile (required, unique)
- birth_date (required)
- password (required)
- password_confirmation (required)
- preferred_language (optional: en|gu|hi)
- user_role (optional: farmer|agro_center)

3. POST /auth/login
Body:
- mobile (required)
- password (required)
- user_role (required: farmer|agro_center)

4. POST /auth/forgot-password/reset
Body:
- mobile (required)
- birth_date (required)
- new_password (required)
- new_password_confirmation (required)

5. POST /auth/logout
Auth required

============================================================
2) PROFILE
============================================================

1. GET /me
Auth required

1.1 GET /me/bills
Auth required (farmer only)
Response item fields:
- id
- bill_date
- payment_status
- amount
- note
- agro_owner_name
- agro_owner_mobile
- bill_photo_path
- bill_photo_url

2. PUT /me
Auth required
Body:
- name (required)
- email (required)
- birth_date (required)
- password (optional)
- password_confirmation (optional)

3. POST /me/profile-image
Auth required
Body (multipart):
- profile_image (required image)

4. PATCH /me/language
Auth required
Body:
- preferred_language (required: en|gu|hi)

5. DELETE /me/all-data
Auth required

6. POST /me/suggestions
Auth required
Body:
- message (required)

============================================================
4) AGRO CENTER MODULE
============================================================

All routes below require:
- logged in user role = agro_center

Dashboard
- GET /agro-center/dashboard

Farmer contacts (name + mobile only)
- GET /agro-center/farmers
- POST /agro-center/farmers
  Body: name, mobile
- PUT /agro-center/farmers/{farmer}
  Body: name, mobile
- DELETE /agro-center/farmers/{farmer}

Bills
- GET /agro-center/bills
  Query optional: farmer_id, payment_status, from_date, to_date
- POST /agro-center/bills
  Body:
  - farmer_id
  - bill_date
  - payment_status (pending|completed)
  - amount (optional)
  - note (optional)
  - bill_photo (required image)
- PUT /agro-center/bills/{agroBill}
  Body same as create, bill_photo optional
- DELETE /agro-center/bills/{agroBill}

Reports
- GET /agro-center/reports
  Query optional: from_date, to_date
